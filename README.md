# Blech Compiler setup: In belch directory:

```
cd blech
dotnet build
dotnet publish -c Release -r osx-arm64 --self-contained
```
For Linux use linux-x64 or linux-arm64, for MacOS use osx-x64 for Intel or osx-arm64 for Apple silicon.
